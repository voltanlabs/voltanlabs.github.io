# VoltanLabs Shared Core

The Shared Core contains reusable code, styles, and UI components used across every VoltanLabs project.

## Planned modules
- js/app-shell.js
- js/navigation.js
- js/storage.js
- js/notifications.js
- js/utilities.js
- js/version.js
- css/studio.css
- components/

## Rules
- New reusable functionality belongs here before project-specific code.
- Creator Suite, Technology Vault, DataByteSprites, and Studio should depend on Shared Core instead of duplicating utilities.
