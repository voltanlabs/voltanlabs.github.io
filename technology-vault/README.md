# VoltanLabs Technology Vault

The Technology Vault is the curated resource library for VoltanLabs, DataByteSprites, Odysseus, and future projects.

## Core rule

Do not adopt a tool just because it is impressive. Adopt it only when it is useful, secure enough, commercially safe, and aligned with our architecture.

## Acceptance stages

1. Security review
2. License review
3. Integration review
4. Performance review
5. Long-term value review

## License ratings

- Green: commercial use appears allowed, usually with license notice/attribution requirements.
- Review: verify the exact license and obligations before using in products.
- Red: research only; do not ship inside monetized VoltanLabs or DataByteSprites products without permission.

## Preferred licenses

- MIT
- Apache 2.0
- BSD

## Review carefully

- MPL
- LGPL
- GPL
- AGPL
- Creative Commons licenses
- Custom licenses

## DataByteSprites rule

DataByteSprites should use free and open-source foundations whenever possible. Code that is non-commercial, unclear, abandoned, or security-risky stays in research only until cleared.

## Website structure

- `/technology-vault/index.html` renders the dashboard.
- `/technology-vault/data/technologies.json` stores the resource database.
- `/technology-vault/js/vault.js` renders searchable cards and statistics.
