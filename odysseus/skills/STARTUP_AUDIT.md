# Startup Audit

## When To Use
At the beginning of every new session.

## Procedure

1. Run pwd.
2. List current directory.
3. Discover workspace folder.
4. Discover repositories.
5. Verify git status.
6. Verify tool availability.
7. Summarize environment.

## Output

- Current path
- Workspace path
- Repository path
- Current branch
- Available tools
- Unknowns requiring investigation
